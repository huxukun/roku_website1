# AR导航系统功能路线图

## 项目概述
基于网页端的AR全息显示骑行导航系统，集成高德地图 API，提供实时导航指引。

---

## 🎯 核心功能架构

### 启动流程（页面加载时）
1. **init()** - 初始化入口：创建迷你地图、箭头旋转、语音模块
2. **_bootstrapNavigationSystem()** - 并行启动：
   - `_loadAMap()` - 加载高德 SDK（Geocoder、AutoComplete、Riding）
   - `navigator.geolocation.getCurrentPosition()` - 获取 GPS 定位
3. **_startGpsWatch()** - 持续监听位置变化，实时更新导航
4. **_reverseGeocode()** - 反地理编码获取当前道路名称

### 设置目的地流程
1. **startNav()** - 显示目的地输入框
2. POI搜索/地址输入 → 用户确认
3. **_applyDestination(name, lng, lat)** - 设置目的地：
   - 清除旧路线状态
   - 同步到迷你地图（显示目的地标记）
   - 启动 GPS + 指南针
4. **_tryPlanRoute()** - 尝试规划路线：
   - 检查 GPS、目的地、Riding 服务是否就绪
   - 调用 `_planRoute()` 获取真实路线
   - 如有条件不满足，每秒重试最多 20 秒
5. **_planRoute()** - 调用 `AMap.Riding.search(起点, 终点)`：
   - 解析返回的 steps（转弯步骤）
   - 生成 fullPath（完整折线坐标）
   - 写入导航状态

### 导航中流程（实时更新）
1. **GPS 位置变化** → `_updateNavProgress(lat, lng)`：
   - `_projectPointToPath()` - 将当前位置投影到路线上，计算累计距离
   - 根据累计距离找到当前所在步骤和下一步
   - 计算箭头方向、距离、转弯动作
2. **同步到 MiniMap** → `setNavProgress()`：
   - 绘制已走/未走路段（不同颜色）
   - 标记下一转弯点位置
3. **信息面板更新**：总距离、预计时间、下一转弯、行驶距离
4. **语音播报**：接近路口时触发转弯提示

### 调试模式（D 键切换）
1. **_setDebugMode(true)** - 关闭 GPS，启用地图点击/拖动控制
2. **自由模式**（默认）：用户完全控制位置和朝向，箭头根据路线重新计算
3. **路线模式**（F 键切换）：沿路线前进，heading 自动跟随
4. **Alt 键**：临时在路线模式下使用自由移动

## 🧪 测试步骤

**标准测试流程**：
1. 打开 `nav-site/index.html`
2. 等待页面加载 → 迷你地图应该显示，状态栏显示"定位中..."
3. 若获取 GPS 成功 → 地图中心对准当前位置，状态栏显示道路名称
4. 点击"开始导航" → 输入目的地（如"天安门"或"国贸"）
5. 从 POI 列表中选择 → 路线规划启动
6. 路线绘制到迷你地图上 → 蓝色折线显示完整骑行路线
7. 箭头显示"直行"或下一个转弯方向 + 距离
8. 按 **D** 进入调试模式 → 在地图上模拟移动
9. 按 **↑↓** 前后移动 → 观察箭头指向变化、路线颜色更新
10. 接近路口时会语音播报（如"前方 100 米右转"）
11. 到达目的地 → 金色"到达"标记 + 语音提示

**调试模式快捷键**：
```
D 键：开关调试模式
F 键：自由模式 / 路线模式切换
↑ W / ↓ S：前进 / 后退
← A / → Q：左偏 / 右偏（旋转朝向）
Shift：加速移动
Alt：临时自由移动（路线模式下按住）
点击迷你地图：直接瞬移到指定位置
拖动迷你地图：边移动边改变朝向
```

## 🔧 数据结构

### 路线步骤 (navSteps[i])
```javascript
{
  index: 0,                    // 步骤序号
  path: [[lng, lat], ...],     // 该段的路径坐标
  distance: 500,               // 该段距离（米）
  instruction: '沿长安街向东骑行500米',
  road: '长安街',              // 道路名称
  action: 'left',              // 动作类型: straight/left/right/uturn/arrive
  turnPoint: [lng, lat],       // 转弯点坐标
  cumDistance: 1000            // 到达该步骤起点的累计距离
}
```

### 完整路径 (navFullPath)
```javascript
[[fromLng, fromLat], [lng1, lat1], [lng2, lat2], ..., [toLng, toLat]]
// 平滑折线路径，沿真实道路行进
```

### 导航状态 (state.*)
```javascript
state.navSteps: [...]            // 所有步骤
state.navFullPath: [[lng,lat],...] // 完整折线路径
state.navTotalDistance: 5000     // 总距离（米）
state.navProgressMeters: 1500    // 当前累计距离
state.navCurrentStepIdx: 2       // 当前所在步骤
state._routePlanned: true        // 是否已规划路线
state._navInitialized: true      // 导航引擎是否就绪
state.debugFreeMode: true        // 当前是否为自由移动模式
```
