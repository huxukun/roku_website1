# 子网站未来需求记录

> 创建日期: 2026-06-10
> 主站风格: 合成波/赛博朋克风格3D作品集

---

## 🎯 方案 A：个人作品集展示子网站

**定位：** 独立的作品展示画廊

**功能模块：**
1. **作品集首页** - 3D网格背景 + 作品卡片瀑布流
2. **作品详情页** - 每个作品有独立页面（大图 + 描述 + 相关推荐）
3. **分类浏览** - 按"3D建模"、"游戏设计"、"视频剪辑"、"编程项目"分类
4. **搜索筛选** - 关键词搜索 + 标签筛选
5. **联系表单** - 合作邀请

**技术特点：**
- 保持主站的霓虹/合成波视觉风格
- 独立域名或子目录部署
- 响应式设计，手机端完美适配

---

## 🎮 方案 B：交互式游戏小站

**定位：** 可玩的在线游戏合集

**功能模块：**
1. **游戏大厅** - 展示你开发的小游戏列表
2. **游戏页面** - 每个游戏可以直接在浏览器玩
3. **排行榜** - 游戏分数记录（本地存储）
4. **操作指南** - 键盘/触屏操作说明

**技术特点：**
- 基于 HTML5 Canvas/WebGL 的轻量游戏
- 霓虹/像素风格统一
- 每个游戏独立页面，可分享链接

---

## 🎨 方案 D：3D互动展示站

**定位：** 沉浸式 3D 体验展示

**功能模块：**
1. **旋转展厅** - 在 3D 空间中浏览你的作品
2. **模型预览器** - 可以旋转查看 3D 模型
3. **粒子动画展示** - 展示你的视觉特效作品
4. **WebGL 实验场** - 各种创意实验项目

**技术特点：**
- 基于 Three.js 的沉浸式体验
- 鼠标/触屏交互
- 加载动画和过渡效果

---

## 📋 通用技术栈

| 组件 | 选型 | 原因 |
|------|------|------|
| **构建工具** | Vite 5.x | 和主站一致，稳定可靠 |
| **3D 引擎** | Three.js | 主站已经在用，复用资源 |
| **样式方案** | 原生 CSS + NES.css | 保持合成波风格统一 |
| **部署** | Vercel | 和主站一致，自动部署 |
| **数据存储** | 本地 JSON 或 Supabase | 简单内容用 JSON，动态内容用 Supabase |
